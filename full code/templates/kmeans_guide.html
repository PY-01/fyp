Welcome to the k-Means Clustering Guide!

This page will walk you through using the k-Means algorithm on our platform, from inputting data to adjusting parameters and visualising results.

Steps to Use k-Means

  1. Input your dataset values (coordinates only, as k-Means doesn’t use labels).
  2. Select the value of k, the number of clusters.
  3. Click "Run" to see the clustered results and visualisation.
    • Buttons:
      • Play: Starts the animation of the k-Means clustering process.
      • Pause: Pauses the animation, allowing you to observe specific steps in the clustering process.
      • Forward: Moves the animation one step forward.
      • Rewind: Moves the animation one step backward.
      • End: Jumps to the final clustered result.
      • Reset: Resets the clustering to the initial state.
    • Radio Buttons (Playback Options):
      • Once: Plays the clustering animation once.
      • Loop: Continuously loops the clustering animation.
      • Reflect: Plays the animation forward and then in reverse.
    • Slider (Animation Speed):
      • Animation Speed: Adjusts the speed of the clustering animation.
